# AI-Powered-Credit-Card-Fraud-Detection
# 💳 AI-Powered Credit Card Fraud Detection System using XGBoost and SMOTE

## 📌 Overview

This project detects fraudulent credit card transactions using Machine Learning techniques. Since the dataset is highly imbalanced, SMOTE (Synthetic Minority Oversampling Technique) is used to balance the classes, and XGBoost is used for classification.

---

## 🚀 Features

- Fraud detection using Machine Learning
- Handles imbalanced data using SMOTE
- High-performance XGBoost classifier
- Data visualization using Matplotlib and Seaborn
- Feature importance analysis
- Confusion matrix visualization
- FinTech and Cybersecurity application
